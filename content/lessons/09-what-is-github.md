## GitHub란 무엇인가?

**GitHub**는 Git 저장소를 온라인에서 호스팅해주는 서비스입니다. 쉽게 말해, 내 컴퓨터에 있는 Git 저장소를 인터넷에 올려두는 공간입니다.

Git과 GitHub는 다릅니다:
- **Git**: 버전 관리를 위한 도구 (소프트웨어)
- **GitHub**: Git 저장소를 호스팅하는 웹 서비스 (플랫폼)

GitHub 외에도 GitLab, Bitbucket 등 유사한 서비스가 있습니다.

## 원격 저장소(Remote Repository) 개념

지금까지는 내 컴퓨터에만 존재하는 **로컬 저장소**를 사용했습니다. GitHub에 저장소를 올리면 **원격 저장소**가 생성됩니다.

원격 저장소의 장점:
- **백업**: 컴퓨터가 망가져도 코드가 안전하게 보관됩니다.
- **협업**: 여러 사람이 같은 저장소에 접근해 함께 작업할 수 있습니다.
- **공유**: 오픈 소스 프로젝트처럼 코드를 전 세계에 공개할 수 있습니다.

## GitHub의 주요 기능

### 저장소 (Repository)
프로젝트의 코드, 이슈, 문서를 한 곳에서 관리합니다.

### Pull Request (PR)
브랜치의 변경사항을 메인 브랜치에 반영해달라는 요청입니다. 코드 리뷰의 핵심 도구입니다.

### Issues
버그 보고, 기능 요청, 할 일 목록을 관리하는 기능입니다.

### Actions
코드가 푸시될 때 자동으로 테스트, 빌드, 배포를 실행하는 CI/CD 도구입니다.

### GitHub Pages
저장소의 파일을 이용해 무료로 웹사이트를 호스팅할 수 있습니다.

## GitHub 계정 만들기

GitHub를 사용하려면 [github.com](https://github.com)에서 무료 계정을 만들어야 합니다. 학생이라면 GitHub Education을 통해 더 많은 혜택을 받을 수 있습니다.
